How many neurons, who decides, and what Layer 4 neurons detect

The full architecture — neuron counts at each stage
The input layer has exactly 4,096 neurons (one per pixel). Every other layer's size is chosen by the engineer. Notice how hidden layers often grow then shrink — wider in the middle to learn more combinations, narrower at output.
How Layer 1 neurons actually work — sliding filters
A neuron does not look at the whole 64×64 image
Each Layer 1 neuron has a tiny 3×3 filter — 9 learnable weights. It slides this filter across every possible position in the image (that's 62×62 = 3,844 positions). At each position it asks: "does my pattern match what's here?" The same neuron, same 9 weights, checks the whole image.
So how many neurons does Layer 1 really have?
If you put 32 filters in Layer 1, you have 32 neurons — each with its own 9-weight 3×3 filter. Each one learns a different edge type. 64 filters = 64 neurons. The count is a hyperparameter you set. Typical first layers: 32 to 128 filters.

The output layer is different — one neuron per class
The very last layer (output) does have one neuron per category — one for "cat", one for "dog", one for "rabbit." But these output neurons don't detect features — they just score how strongly the deep features match their class. The feature detection happens in the hidden layers.
