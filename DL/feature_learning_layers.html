How deep learning layers automatically learn features

Click any layer to see what it learns — using a cat photo as example
IN
Raw input Pixel values
The network sees nothing but a grid of numbers — each pixel's brightness (0=black, 255=white). No meaning yet.
255, 240, 18, 0, 12... just numbers
A 64×64 cat photo = 4,096 pixels. Each pixel is just a number. The network has no idea these numbers represent a cat — yet. This is all that goes in.
L1
Layer 1 Edges and gradients
Neurons here fire when they detect abrupt changes in brightness — the outline of something. Very simple. Very local.
vertical edge horizontal edge diagonal line colour change
Each neuron has learned a small filter — like a tiny 3×3 stamp — that it slides across the image. When its stamp matches what's in the image (e.g. a sharp brightness change), it fires. This happens automatically through training. Nobody programmed "look for edges."

Think of it like: the neuron stumbled across edges being a useful signal and latched on.
L2
Layer 2 Shapes and textures
Neurons here combine edges from Layer 1 into bigger patterns — curves, corners, fuzzy textures, circular blobs.
curve corner fur texture circular blob striped pattern
Layer 2 never sees the raw pixels. It only sees what Layer 1 reported. So it's building on top of edges: "if there's a curve here AND a curve there, that's worth noticing." Multiple edges curving around each other = a circle. Two edges meeting at 90° = a corner.

This is why depth matters — each layer can represent something more complex than the one below.
L3
Layer 3 Parts of objects
Now combining shapes into recognisable object parts — eyes, ears, snouts, paws. Getting closer to "cat-ness."
eye shape pointed ear whisker nose shape paw outline
A "pointed ear" detector fires when it sees a triangular cluster of edges arranged the right way. A "whisker" detector fires when it sees a set of thin horizontal lines radiating outward. These neurons never explicitly knew what a cat ear was — they discovered that detecting it helped reduce the loss.

Fascinatingly, if you trained on dogs instead of cats, Layer 3 would learn floppy ears and snouts. The features that emerge depend entirely on what the network was trained to predict.
L4
Layer 4 Whole concepts
The deepest layer combines parts into complete ideas — "cat face," "dog body," "car front." Abstract, high-level understanding.
cat face dog body wheel + hood = car two eyes + nose = face
By this point, a single neuron might fire only for "cat faces seen from the front." Another fires only for "cat faces from the side." These are incredibly specific detectors that emerged purely from showing the network labelled examples and nudging weights.

This is also why transfer learning works — these deep feature detectors for "eyes" and "fur" are useful for recognising dogs, foxes, or any furry animal. The early layers transfer almost perfectly to new tasks.
OUT
Output Final answer
One neuron per class. Whichever fires strongest wins. Sigmoid or Softmax squashes the scores into probabilities.
cat: 94% dog: 4% rabbit: 2%
Softmax turns raw scores into probabilities that sum to 100%. The class with the highest probability is the prediction. If it's wrong, the loss is high, backpropagation fires, and every weight across all layers gets nudged. Over millions of examples, the weights settle into values that make these probabilities accurate.
